# 网易云音乐歌词转换工具
用于将网易云下载的lrc歌词转换为标准格式

### 主要特征

- 支持歌词批量处理
- 支持自动转为 UTF-8 
- 兼容各大主流系统平台

### 使用方式

1. 下载并安装python解释器  [官方网站>>>](https://www.python.org/)
2. 下载本项目 （特别是“LrcConverter.py”）
3. 在脚步所在目录新建目录 “input”，放入lrc歌词文件
4. 在脚步所在目录打开终端，执行 `python LrcConverter.py`
