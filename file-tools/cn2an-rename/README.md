# 文件重命名 - 中文序号转数字
用于将文件中的中文形式的序号命名为阿拉伯数字序号。

### 使用效果

|      原文件名       |     重命名后      |
| :-----------------: | :---------------: |
|   第二次作业.pdf    |  第02次作业.pdf   |
|  第十五次作业.jpg   |  第15次作业.jpg   |
| ABC第十八次某某.xxx | ABC第18次某某.xxx |

### 使用方式

+ 根据实际需求修改代码，核心是用 `re.search` 进行分组匹配。

+ 把 py 文件放在与待重命名的文件相同的目录下

+ 运行
