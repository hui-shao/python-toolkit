# 文本 UTF-8 和 GBK 转换工具
用于文本文件中 utf-8 编码与 gbk 编码的互相转换

### 主要特征

- 支持文本批量处理
- 支持逐个文件选择转换模式
- 支持自动判断主流系统平台

### 使用方式

1. 下载并安装python解释器  [官方网站>>>](https://www.python.org/)
2. 下载本项目 （特别是“gbk-utf8-converter.py”）
3. 在脚本所在目录新建目录 “input”，放入txt文件
4. 在脚本所在目录打开终端，执行 `python gbk-utf8-converter.py`
